Detailed Description

Expression class declaration. Represents a string that could be evaluated.

An Expression literal is a Literal composed of Numeric, Atom, Complex, and operands between brackets. Infix notation is used in an Expression. It can be evaluated with the operand EVAL. Ex: '(3+2)*5' EVAL gives 3 2 + 5 * = 30.
